Guide d'utilisation du simulateur de lecteur à code barre de l'équipe 4

Après avoir dézippé  et ouvert le dossier simulateur dans matlab, l'utilisateur 
doit tout d'abord ouvrir le fichier "simulation.slx" et démarrer la simulation 
afin que le simulink en question soit chargé dans l'outil CAO.

Ensuite, l'utilisateur doit ouvrir l'application "simulateur.mlapp".Il s'agit de 
la page principale qui ouvrira toutes les pages sous-jacentes.

"simulateur.mlapp" et "simulation.slx" sont les seuls fichiers que l'utilisateur 
aura besoin d'ouvrir durant tout le processus. Les autres fichiers peuvent être
consultées au bon vouloir de l'utilisateur.


Pour lancer une simulation, l'utilisateur doit tout d'abord générer un code barre 
et paramétrer le laser qui sera envoyée sur le code barre en cliquant sur les boutons
"modifier" associés à chacun. Ces deux étapes doivent être obligatoirement réalisées 
à chaque ouverture de l'outil CAO avant de cliquer sur le bouton "démarrer la simulation".

Des valeurs par défaut ont été placées dans chaque bloc de paramètres. Valeurs que 
l'utilisateur peut modifier. Lorsque l'utilisateur démarre une simulation, il doit 
attendre que le voyant "en cours d'exécution" redevient vert avant de cliquer sur les 
boutons associés aux graphes résultants. Par précaution, ces boutons ont été désactivés
lors de l'exécution de la simulation.

L'utilisateur peut sauvegarder et charger les données de tous les paramètres modifiables 
du simulateur. Ainsi que les graphes résultants d'une simulation.
